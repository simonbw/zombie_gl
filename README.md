zombie_gl
=========

ZombieGL is a top-down, 3d, zombie shooter using HTML5 technologies including WebGL (through Three.js), gamepad support, and the Web Audio API.

## Controls
### XBox Controller
  * Left Stick -- Move
  * Right Stick -- Aim
  * Right Trigger -- Shoot
  * X -- Reload
  * Y -- Toggle Flashlight

### Standard
  * Mouse -- Aim
  * Primary Mouse Button -- Fire
  * W -- Move Up
  * S -- Move Down
  * A -- Move Left
  * D -- Move Right
  * R -- Reload
  * F -- Toggle Flashlight
  * G -- Toggle Gamepad
